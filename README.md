> 自动生成竞赛答卷相关文件的工具。支持从 `.docx` 赛题文档中提取小题结构，生成带公式的批改工作簿。

## Quick Start

### 环境要求

- Python >= 3.10

### 安装依赖

- 方式一：使用 uv

```bash
uv sync
```

- 方式二：使用 pip

```bash
pip install .
```

### 准备试题文档

1. 在项目根目录下创建 `doc/`。
2. 将所有 `.docx` 试题文件放入 `doc/` 中。
3. 试题文档需遵循模板要求：
   - 使用 Word 多级列表（自动编号），最终会被解析为 `1-1`、`2-1-1` 等形式。
   - 每道小题末尾附带分值，格式为中文括号，例如“（5分）”，且分值括号必须位于行尾。

### 运行脚本生成表格

```bash
# 使用 uv
uv run generate_xlsx.py

# 或直接使用 Python
python generate_xlsx.py
```

脚本执行后，`dist/` 目录下会生成 `final.xlsx`，即为批改表格。

生成表格包含以下部分：

| Sheet 名称 | 用途 |
|-----------|------|
| **答卷基本信息** | 填写用户名/ID、选题情况、查看批改完成状态，附有使用说明 |
| **各题分表** | 每题一个 sheet，自动填入小题号与满分，得分处可手动填写 |
| **总分统计** | 自动计算全卷总分、最高三题总分及排名 |
